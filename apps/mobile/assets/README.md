# Assets Directory

This directory should contain the following image assets:

- `icon.png` - 1024x1024 app icon
- `splash.png` - 1242x2436 splash screen
- `adaptive-icon.png` - 1024x1024 Android adaptive icon
- `favicon.png` - 48x48 web favicon

See `../ASSETS.md` for detailed specifications and creation instructions.

## Temporary Placeholders

Until proper assets are created, the app will use default Expo assets. To create quick placeholders, run:

```bash
# From the mobile app directory
bash ../scripts/create-placeholder-assets.sh
```
